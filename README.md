# DataCamp Courses Practice

This repository collects practice code and mini exercises completed while following DataCamp courses.  
The goal is to keep all learning work in one place, with a clear structure by technology and topic.

## Contents

- `SnowFlake_SQL/` – Snowflake SQL practice (CTEs, CASE expressions, joins and query patterns).

## Snowflake SQL

The `SnowFlake_SQL/sql` folder currently includes:

- `cte_artist_listening_time.sql` – total listening time per artist based on invoice line items.
- `case_greatest_hits_album_sales.sql` – revenue analysis for albums flagged as "Greatest Hits".
- `case_track_metadata_quality.sql` – data quality checks on track metadata (composer vs. artist).
- `case_customer_churn_high_salary.sql` – classifies customers as 'Churned' or 'Active' and analyzes churn among high-salary customers.

More folders will be added over time as new DataCamp courses are completed.
